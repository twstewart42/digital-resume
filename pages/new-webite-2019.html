A New Job, A New Website, and A New Cup of Coffee

New Website

Welcome internet travelers, to the new edition of my public website. For the 2019 edition, I decided to go as minimal as possible, as a way to fight back against the endless noise that the majority of the internet has become. Videos and advertisements attack us at every click, endless scroll refuses to let us leave, but here in this small corner of the web, I thought it might be nice to give readers a break from all that activity. A return to basics and normalcy.

The previous version of this website was running on a simple wordpress installation, hosted on GCP with cloudflare acting as a CDN, SSL via Let's Encrypt. Fun, and neat, but it was costing me like $5 a month for the smallest sliver of a server GCP would provide, and as time went on it became clear that I did not need all these extra wordpress features. The site was nothing more than text and some fancy menus. With AWS's free tiers and getting out side of the everything-must-run-on-a-machine mindset, I was able to setup this website using Terraform to do the following things, host files out of S3, then leveraged CloudFront for SSL termination and act as CDN. If I've done my math right everything should be FREEfor a year and after that pennies in total cost. For heavier use websites, this might not be the way to go, but for me and this website, this setup is perfect.

All of the code for both the website and deploying the technologies via terraform is available on this github repo. So if you want, you can build your own digital resume or personal website following this as an example.
